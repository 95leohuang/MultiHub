Mermaid 圖表

                  
支援流程圖、圓餅圖、甘特圖等多種強大圖表：

                  
```mermaid
graph TD
  A[開始] --> B{判斷}
  B -->|是| C[成功]
  B -->|否| D[失敗]
```

                  
```mermaid
pie title 支援圖表
  "流程圖" : 40
  "圓餅圖" : 20
  "序列圖" : 20
  "其它" : 20
```
